# 妖气山视频管理系统

#### 项目介绍
1. 第一款面向大众的java版的视频管理系统。
2. 妖气山视频管理系统理论上可以制作任何类型的视频网站。

#### 软件架构
1. 前端：javaex
2. 后端：ssm
3. 数据库：sql server 2012以上
4. 编译器：eclipse
5. JDK：1.8
6. tomcat：tomcat8
